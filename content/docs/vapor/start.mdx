## 主题设置

进入 **“网站后台/主题”** ，你会看到有若干个 tab 页可以进行切换

### 示例配置

<Tabs items={['首页介绍词', '博主介绍关键词']}>
  <Tab value="首页介绍词">
    <Callout type="info">
      提示：介绍词中的 **a标签** 会被主题格式化成独特的样式。
    </Callout>
		```javascript title="example"
		  我是 <a rel="noopener noreferrer" target="_blank" href="https://blog.scott-studio.cn/">三禾木木</a>，不知名前端攻城狮，分享编程经验和技术干货。
		```
	</Tab>
  <Tab value="博主介绍关键词">

    <Callout type="info">
      提示：首页的 **博主介绍关键词** 分为4个词语，使用英文逗号分隔。
    </Callout>

    ```javascript title="example"
      开发者,行动派,践行者,探索者
    ```
  </Tab>
</Tabs>

## 文章封面

- 建议封面尺寸为16:9
- 推荐使用 [unsplash.com](https://unsplash.com/) 找到你喜欢的图片作为文章封面
