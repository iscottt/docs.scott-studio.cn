import { Step, Steps } from 'fumadocs-ui/components/steps';

<Callout title="提示">
友情链接其实本质上也是一个页面，Somnia主题中需要在页面的右侧，选择 “设置为友情链接页面”
</Callout>
## 添加页面

<Steps>
<Step>
进入“网站后台/页面/新建页面”
</Step>
<Step>
标题输入：友情链接（或其他你喜欢的标题）
</Step>
<Step>
内容输入友情链接显示给访客的内容（例如：友链申请规则）
</Step>
<Step>
边栏的“评论”窗格，勾选“✅启用评论”
</Step>
<Step>
保存并发布
</Step>
</Steps>

## 添加友链

<Callout title="提示">
不要忘记选择友链分类
</Callout>
<Steps>
<Step>
进入“网站后台/友情链接/添加友链
</Step>
<Step>
标题输入：友链名称
</Step>
<Step>
站点地址输入：友链地址
</Step>
<Step>
站点描述输入：友链描述
</Step>
<Step>
站点图片：友链图片
</Step>
</Steps>


## 添加友链分类

<Callout type="warning" title="注意">
友情链接只有选择了分类才会以分类列表的形式展示，否则不会展示
</Callout>

<Steps>
<Step>
进入“网站后台/友情链接/友链分类
</Step>
<Step>
点击右上角的`加号`
</Step>
<Step>
输入名称
</Step>
<Step>
输入别名
</Step>
<Step>
保存
</Step>
</Steps>
